ORA-01578: разрушен блок данных ORACLE (файл # 4, блок # 935432)

Ищем объект в котором сбойный блок:

SELECT owner, segment_name, segment_type
FROM dba_extents
WHERE file_id = 4
AND 935432 BETWEEN block_id AND block_id + blocks - 1;

Объект I_ENV_LOCK_SESSION_UNIT —  индекс, то можно пересоздать:

drop index I_ENV_LOCK_SESSION_UNIT;
create index I_ENV_LOCK_SESSION_UNIT on ENV_LOCK( SESSION_ID,UNITCODE );
