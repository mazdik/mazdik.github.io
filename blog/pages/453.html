Переход с APEX Embedded PL/SQL Gateway в APEX Listener

После установки APEX

Отключить Oracle XML DB Protocol Server
EXEC DBMS_XDB.SETHTTPPORT(0);

ALTER USER APEX_PUBLIC_USER ACCOUNT UNLOCK;
ALTER USER APEX_PUBLIC_USER IDENTIFIED BY new_password;

Настройка RESTful Services

@apex_rest_config.sql
Будут созданы два пользователя: APEX_LISTENER и APEX_REST_PUBLIC_USER
Скачать http://www.oracle.com/technetwork/developer-tools/apex-listener/downloads/index.html

Запуск APEX Listener в Standalone Mode
java -jar apex.war

Standalone Mode подходит только для использования в целях развития, и не поддерживается для использования в развертывании производства.
Используйте поддерживаемый сервер приложений Java EE для развертывания производства.
