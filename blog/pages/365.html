ORA-27478 Задание выполняется и не останавливается scheduler job oracle

Unable to stop running scheduler job

ORA-27478 Задание выполняется и не останавливается.

Решение: убить сессии.

ALTER SYSTEM KILL SESSION ‘sid,serial#’;
ALTER SYSTEM KILL SESSION ‘1347,34538’;

ALTER SYSTEM DISCONNECT SESSION ‘sid,serial#’ IMMEDIATE;
ALTER SYSTEM DISCONNECT SESSION ‘1347,34538’ IMMEDIATE;

sid и serial можно посмотреть через PL/SQL develpoer -> Tools -> Sessions
