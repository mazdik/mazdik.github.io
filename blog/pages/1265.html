Angular 2 CRUD table using Yii2 RESTful API and ORDS

Simple CRUD table component for Angular 2 using REST backend and Bootstrap 3 CSS. (Demo)
Supports Yii2 RESTful API and ORDS (Oracle REST Data Services). GitHib

Features

  • Filtering
  • Sorting
  • Pagination
  • Modal
  • Create/Update/Delete
  • Loading indicator
  • MultiSelect
  • Scrolling with fixed header horizontally and vertically
  • Frozen columns
