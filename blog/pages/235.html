Удалить Label Securitie Option

При перекомпиляции ругается на LBACSYS.LBAC_EVENTS (Label Securitie Option)

Просмотр объектов Label Securitie Option:

SELECT
object_type
,COUNT(*)
FROM dba_objects
WHERE owner='LBACSYS'
GROUP BY object_type;

Удаляет Label Securitie Option с экземпляра базы (в режиме startup upgrade):
@C:\oracle\product\11.2\dbhome\RDBMS\ADMIN\catnools.sql

Статусы компонентов Oracle:

select comp_id, comp_name, version, status, schema
from dba_registry;

Восстановление Oracle Database Packages and Types:

startup migrate;
SQL> @?/rdbms/admin/catalog.sql
SQL> @?/rdbms/admin/catproc.sql

catalog.sql будет пересоздан Oracle database Catalog Views component
catproc.sql будет пересоздан Oracle database Packages and Types component
