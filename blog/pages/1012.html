Log sequence in archive filename wrapped

ARC0: Warning. Log sequence in archive filename wrapped
to fix length as indicated by %S in LOG_ARCHIVE_FORMAT.
Old log archive with same name might be overwritten.

Старые архивные журналы с таким же названием могут быть перезаписаны.

%S Сиквенс журнала. Это число дополняется слева нулями. Длина 5 символов.
select max(sequence#) from v$archived_log;
157219
ARC57219_0730778068.001

%s Сиквенс журнала. Это число не дополняется слева нулями. Длина без ограничения.

Решение:
вместо:
log_archive_format = ARC_%S_%R_%T.ARC
сделать:
log_archive_format = ARC_%s_%R_%T.ARC

ALTER SYSTEM SET log_archive_format = ‘arch_%r_%t_%s.arc’ SCOPE=spfile;
