LOCAL_LISTENER

LOCAL_LISTENER указывает имя сети, что разрешает адрес или список адресов Oracle Net местных слушателей (то есть listeners, которые работают на той же машине). Адрес или список адресов, указанных в файле TNSNAMES.ORA.
tnsnames.ora

TESTDB =
(DESCRIPTION =
(ADDRESS = (PROTOCOL = TCP)(HOST = localhost)(PORT = 1521))
(CONNECT_DATA =
(SERVER = DEDICATED)
(SERVICE_NAME = testdb)
)
)

Для init.ora:
LOCAL_LISTENER=TESTDB

Для spfile:
ALTER SYSTEM SET LOCAL_LISTENER=TESTDB

или создать spfile из init.ora

create spfile=’/u01/app/oracle/product/11.2.0/db_1/dbs/spfiletestdb.ora’  FROM pfile=’/u01/app/oracle/admin/testdb/pfile/init.ora‘;

показать параметр:
show parameter local_listener
