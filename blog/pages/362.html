rlwrap — история команд в sqlplus и rman в linux

rlwrap — пакет, который позволяет хранить историю команд в SQL*PLUS и RMAN в Linux (его необходимо прописывать отдельной строкой в bash профиле).
Установив данный пакет, вы сможете использовать кнопки вверх, вниз для просмотра истории введенных команд, правильную работу команды backspace и др.

yum install -y \
readline-devel.x86_64

# cd /tmp
# wget http://utopia.knoware.nl/~hlub/uck/rlwrap/rlwrap-0.37.tar.gz

# tar zxvf rlwrap-0.37.tar.gz
# cd rlwrap-0.37
# ./configure
# make
# make check
# make install

Прописать алиасы в ~/.bashrc в виде:

# User specific aliases and functions
alias sqlplus=»rlwrap sqlplus»
alias rman=»rlwrap rman»

При запуске rlwrap создается файл истории ‘~/.command_history’, где command — название программы которую вы запускаете.
