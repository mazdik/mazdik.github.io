Веб-приложение DBA STAT (APEX)

DBA STAT — мое первое веб-приложение на основе APEX для сбора статистики по базам Oracle

С помощью dblink’ов собираются следующие данные:
— Имя сервера (sql запрос в настройках)
— Общий размер дата файлов базы
— Полный размер бд = Общий размер бэкапов рмана + общий размер архивлогов + общий размер дата файлов базы
— Дата последнего бэкапа RMAN
— Размер бэкапа RMAN
— FRA использовано
— FRA общий размер
— Размер дата файла табличного пространства (названия добавляются в настройках)
— Процент использования табличного пространства
— Данные с пользовательской процедуры (указывается в настройках)
Остальное лень расписывать.
В 18 версии добавлена многопоточность. Количество потоков указывается в настройках.
