DBVERIFY

DBVERIFY — утилита командной строки, которая выполняет проверку целостности файлов базы данных.
Эта утилита только для файлов данных, с control и redolog файлами не будет работать.

dbv file=E:\FRS\PARUS_MAIN.DAT logfile= dbv.log blocksize=8192 feedback=100

dbv file=E:\FRS\PARUS_MAIN.DAT — файла данных PARUS_MAIN.DAT
logfile= dbv.log — лог файл dbv.log
blocksize=8192 — размер блока 8192
feedback=100 — прогресс в командной строке в виде одной точки (.)
