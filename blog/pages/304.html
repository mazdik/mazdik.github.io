NLS_LANG при импорте базы Oracle с Linix

Перенос Oracla с Linux на Windows.

База данных Oracle на Linux с кодировкой NLS_CHARACTERSET = CL8ISO8859P5

Новая база данных Oracle на Windows с кодировкой NLS_CHARACTERSET = CL8MSWIN1251

При экспорте и импорте через Windows в regedit прописать NLS_LANG  = AMERICAN_AMERICA.CL8ISO8859P5

После импорта дампа прописать NLS_LANG  =  RUSSIAN_RUSSIA.CL8MSWIN1251
