Apex GlassFish server

Создаем файл i.war:
java -jar apex.war static D:\apex\images

Устанавливаем GlassFish server:
D:\glassfish-4.0-windows.exe -j «C:\Program Files\Java\jdk1.7.0_51»

Заходим в админку:
http://localhost:4848

Application -> Deploy -> выбрать apex.war

Type: Web Application
Context Root: apex
Application Name: apex
Status: Enabled
Description: Application Express Listener
Все остальные значения по умолчанию и нажмите кнопку ОК.

Application -> Deploy -> i.war

Закинуть настройки сделанные ранее (java -jar apex.war)
C:\glassfish4\glassfish\domains\domain1\config\listener.conf
