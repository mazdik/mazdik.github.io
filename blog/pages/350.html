Переход Oracle от Standard Edition к Enterprise Edition

Убедитесь, что номер версии вашей Oracle Standard Edition такой же, как выпуск Oracle Enterprise Edition. Например, если Standard Edition версии 11.2.0.2, то следует обновить версией 11.2.0.2 Enterprise Edition .

1. Остановите все службы Oracle.

2. Сохраните
$ORACLE_HOME/network/admin
$ORACLE_HOME/database/ (На Linux  $ORACLE_HOME/dbs/)
ветку реестра HKEY_LOCAL_MACHINE\SOFTWARE\ORACLE

3. Удалите софт с помощью Oracle Universal Installer, оставив только Oracle_Home
Удалите ветку реестра HKEY_LOCAL_MACHINE\SOFTWARE\ORACLE

4. Установите Enterprise Edition

5. Запустите базу данных.

в regedit добавить SID по умолчанию «ORACLE_SID»=»MURGB»

SQL> select * from v$version;

PL/SQL Release 11.2.0.2.0 — Production
CORE    11.2.0.2.0      Production
