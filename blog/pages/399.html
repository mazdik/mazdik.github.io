Создание excel файла с помощью plsql

С помощью этого пакета очень легко создать файл Excel 2007 за несколько строк PL / SQL кода.

begin
as_xlsx.query2sheet( 'select * from dual' );
as_xlsx.save( 'MY_DIR', 'my.xlsx' );
end;
