DATABASE LINK

Просмотреть все имеющиеся ссылки подключений:

select * from dba_db_links;

Создать ссылку подключения к удаленной базе данных:

create database link testlink_db1 connect to system identified by sys
using
‘(DESCRIPTION=
(ADDRESS=
(PROTOCOL=TCP)
(HOST=192.168.2.127)
(PORT=1521))
(CONNECT_DATA=
(SID=TESTDB)))’;

или

create database link testlink_db2 connect to system identified by sys
using
‘192.168.2.127:1521/TESTDB’;

SQL запрос просмотра сессий удаленной базы данных:

select * from v$version@testlink_db1;

Удалить линк подключения:

drop database link testlink_db1;
