Скрипты для запуска Observer

Под Windows

observer.cmd:

set ORACLE_HOME=C:\app\product\11.2.0\dbhome_1
call dgmgrl -silent sys/sys@MURGBPROD "stop observer"
call dgmgrl -silent sys/sys@MURGBPROD "start observer"

Под Linux

observer.sh:

./observer_loop.sh

observer_loop.sh:

dgmgrl << edgmgrl
connect sys/pass@SID
STOP OBSERVER;
START OBSERVER;
edgmgrl
